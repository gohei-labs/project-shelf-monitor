ドキュメントシステム仕様の移行

これは Issue #42 の最初の移行スライスです。ドキュメントシステム仕様を、英語版と日本語版の対応ルートへ移します。

承認してほしいこと

  1. 英語版仕様の場所を docs/en/specs/documentation-system.md にする。
  2. 日本語版仕様の場所を docs/ja/specs/documentation-system.md にする。
  3. 旧パスの詳しい内容を保持したうえで、旧パスを削除する。
  4. 旧配置 Markdown のバックログを66件から64件に減らす。
