共有の visible agent manifest location として agent-context/ を承認し、不要になった secret-scan allowlist の削除を承認し、root hidden path には tool または ecosystem reason を必須にする方針を承認する。

推奨

リスクと軽減策

リスク軽減策
agent が古い hidden manifest を探し続ける。AGENTS.md、CLAUDE.md、GEMINI.md を追加し、すべて agent-context/agent-core-context.json へ誘導する。
future contributor が慣れで project-specific dotfile を追加する。scripts/guardrails/check-root-hidden-paths.ps1 が、明示許可されていない path を失敗させる。
local secrets と tracked control files が混同される。方針では .env のような ignored local runtime files を許可しつつ、tracked または unignored の hidden control paths をブロックする。
