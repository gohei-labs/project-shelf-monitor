Review 0020: Docs Root Language Structure

Requested Review Decision

  1. Approve making docs/en and docs/ja the only top-level directories under docs.
  2. Approve moving review artifacts to docs/en/reviews and docs/ja/reviews.
  3. Approve moving the agent context manifest from docs/agent-core-context.json to .agents/agent-core-context.json.

Why This Change Exists

The previous PR scope migrated only one specification pair. That left legacy Markdown and review folders directly under docs, even though Docusaurus uses docs as the source tree. This made the rendered documentation reflect an inconsistent source structure.
